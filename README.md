# SkillSwap 🚀

SkillSwap is a simple platform where people can **connect, learn, and help each other by exchanging skills**.

The idea is straightforward — everyone is good at something, so why not use that to learn something new from someone else?

---

## 💡 What you can do here

* Create your profile and list your skills
* Discover people who have skills you want to learn
* Chat and connect with them
* Request skill exchanges
* Earn **skill points** based on your activity
* See where you stand on the **leaderboard**

---

## ⚙️ How to Run Locally

### 1. Clone the repo

```
git clone https://github.com/AddhyanB/SkillSwap.git
cd SkillSwap
```

---

### 2. Start Backend

```
cd server
npm install
npm run dev
```

Create a `.env` file in the server folder:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

---

### 3. Start Frontend

```
cd client
npm install
npm run dev
```

---

## 🏆 Leaderboard

Users earn **skill points** when they:

* Help others
* Complete skill exchanges
* Stay active

Leaderboard simply ranks users based on these points.
